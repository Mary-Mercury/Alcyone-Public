Публичный репозиторий дипломной работы, Alcyone. Приложение разработанное с целью помочь студентам легче ориентироваться в расписании. Версия на момент написания README - 1.2.2.

Используемые инструменты:
- Kotlin
- Android Studio
- MVVM
- Hilt
- Supabase
